# Parameterized Wing Modeling Tool based on CATIA VBA
基于 CATIA VBA 二次开发的机翼参数化建模插件，支持导入本地翼型坐标文件，通过可视化界面设置机翼几何参数，一键生成参数化机翼三维模型。

## ✨ 功能特性
- **翼型文件导入**：支持浏览本地翼型坐标文件，自动读取翼型轮廓数据并完成基准曲线构建
- **全参数化几何定义**：可自定义翼展、根/梢弦长、后掠角、上反角、扭转角等核心几何参数
- **一键自动化建模**：参数确认后自动完成曲面扫掠、实体生成等全流程建模操作
- **轻量化交互界面**：内置可视化操作面板，参数调整直观便捷，无需复杂 CATIA 操作基础

## 📋 环境要求
- 操作系统：Windows 10 / Windows 11
- 软件版本：CATIA V5 R20 及以上版本（需启用 VBA 宏功能）
- 翼型文件：支持标准二维翼型坐标数据文件（.dat 等通用格式）

## 🚀 安装与加载步骤
1. **下载插件文件**
   将 `wing.catvba` 文件下载保存至本地电脑，建议存放路径不含中文与特殊字符。

2. **添加 VBA 宏库**
   1. 打开 CATIA 软件
   2. 依次点击顶部菜单栏：`工具` → `宏` → `宏`
   3. 在弹出的宏窗口中，点击「宏库」按钮
   4. 库类型选择「VBA 项目」，点击「添加现有库」
   5. 在文件浏览器中选中本地保存的 `wing.catvba` 文件，确认完成库添加

3. **运行插件**
   按下快捷键 `Alt + F8` 唤出宏运行窗口，选择对应宏程序，点击「运行」即可启动建模工具界面。

## 📖 使用流程
1. 启动插件后，点击「浏览」按钮选择本地翼型坐标文件
2. 在参数面板中依次设置机翼的各项几何参数
3. 确认参数无误后，点击「生成机翼曲面」按钮
4. 等待程序自动完成建模，生成完毕后可在 CATIA 中查看、编辑与导出模型

## ⚙️ 核心可调参数
| 参数名称 | 参数说明 | 单位 |
|---------|----------|------|
| 翼展 | 机翼展向总长度 | mm |
| 根弦长 | 机翼根部弦向长度 | mm |
| 梢弦长 | 机翼梢部弦向长度 | mm |
| 后掠角 | 机翼前缘后掠角度 | ° |
| 上反角 | 机翼横向上反角度 | ° |
| 扭转角 | 翼梢相对翼根的扭转角度 | ° |

## 📁 文件说明
- `wing.catvba`：插件核心程序文件，包含全部建模逻辑与交互界面代码
- 翼型数据文件：用户自备的二维翼型坐标文件，推荐采用 Selig 格式 .dat 文件

## ⚠️ 注意事项
1. 请确保 CATIA 已启用宏执行权限
2. 插件存放路径与翼型文件路径建议使用英文命名，避免中文路径导致的读取异常
3. 生成模型前请保存当前 CATIA 文档，避免意外操作导致数据丢失
4. 请在合理范围内设置几何参数，极端参数可能导致曲面构建失败
